01. Create an ASP.NET Web Form, which prints the type of the browser and the client IP address requested .aspx page.
02. Create a ASP.NET Web Form which appends the input of a text field when a button is clicked in the Session object and then prints it in a <asp:Label> control. Use List<string> to keep all the text lines entered in the page during the session lifetime.

05. Implement a graphical Web counter. It should display as JPEG image the total number of visitors of the requested .aspx page since the start of the Web application. Keep the number of visitors in the Application object. What happens when the Web server is stopped?
06. Re-implement the previous task to keep the total number of visitors in SQL Server database.
