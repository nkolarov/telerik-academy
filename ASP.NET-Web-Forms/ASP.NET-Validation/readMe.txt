01. Create a form to register users with fields for preferred user name, password, repeat password, first name, last name, age, email, local address, phone and an “I accept” option. All fields are required. Valid age is between 18 and 81. Display error messages in a ValidationSummary. Use a regular expression for the email and phone fields.
02. Separate the fields in groups and validate them using Validation Groups. The Validation Groups should be at least three – Logon Info, Personal Info, Address Info.
